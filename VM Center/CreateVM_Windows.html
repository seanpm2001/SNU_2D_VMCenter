Copyright update issue - 3.28.2019

Sean found out from winworldpc.com that Microsoft still holds copyright ground on Windows XP and above. Originally, I thought that Windows XP and Vista were OK to add to our virtual machine collection, but they still have support.

Since no ISO images have even been added here yet, we are fine. The beta of Windows XP is available though, Microsoft allows that. We probably aren't even ready for Windows XP yet, as that would take a ton of memory
